Configurer le serveur

Le bot a besoin d'un channel d'insultes, le nom par défaut est général mais vous pouvez le modifier via la variable INSULT_CHANNEL.

Variables d'environnements du bot

  • APP_TOKEN token de connexion au bot
  • INSULT_CHANNEL nom du channel où seront envoyées les insultes
  • INSULT_TIME_SPACE intervale de temps (en secondes) entre deux insultes

COmmandes

  • insult! insulte au hasard un membre du serveur ou un politique français actuel
  • setTimeSpace [number] définis l'intervale de temps (en secondes) entre deux insultes. La valeur par défaut est la variable d'environnement INSULT_TIME_SPACE
